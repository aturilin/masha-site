# Maria Brodskaya - Psychologist Website

## Overview
This is a static website for Maria Brodskaya, an existential and CBT psychologist. The site is built with HTML5, CSS3, and Vanilla JavaScript.

## Project Structure
- `index.html` - Main HTML structure
- `style.css` - Styling and layout
- `script.js` - JavaScript functionality (smooth scrolling)
- `server.py` - Python HTTP server for local development
- `PLAN.md` - Detailed implementation plan
- Images: 3 photos and 1 screenshot

## Technology Stack
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Server**: Python HTTP server (development)
- **Port**: 5000
